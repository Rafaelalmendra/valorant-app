## 💻 Project

Valorant Guide is intended to show valorant information interactively for novice players.

<br>

## 🧪 Technologies

- Nextjs
- Styled Components
- Typescript
- Axios
- Valorant API

<br>

## 🚀 How to run

To start it, follow the steps below:

```bash
# Install dependencies
$ yarn

# Start the project
$ yarn dev
```
